# Memability/Memz.co Export Viewer

This small webapp lets you view your [Memability/Memz.co](https://www.memz.co) notes in all their hierarchical glory.

To use this, first download a JSON of your Memability/Memz.co notes from [Google Tasks](https://tasks.google.com/tasks), and place it in this project's root directory with the name `Memability.json`.

Next, simply run `server.sh` and navigate to `http://localhost:8000` in your web-browser.

Your notes should display in all their glory!
