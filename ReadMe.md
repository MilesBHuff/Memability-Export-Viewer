# Memability Export Viewer

This small webapp lets you view your Memability notes in all their hierarchical glory.

To use this, first download a JSON of your Memability notes from Google Tasks, and place it in this project's root directory with the name `Memability.json`.

Next, simply run `server.sh` and navigate to `http://localhost:8000` in your web-browser.

Your notes should display in all their glory!
